# Roger's Research Protocol

## When Researching Topics

Always include these platform filters in searches:

```
site:github.com
site:arxiv.org
site:huggingface.co
site:reddit.com
site:youtube.com
site:x.com
site:twitter.com
site:news.ycombinator.com
```

## Priority Order

1. GitHub - Code and repos
2. arXiv - Research papers
3. Reddit - Discussions
4. YouTube - Visual content
5. X/Twitter - Real-time updates

## Rule

When Dan asks to "research" - always use platform filters above.
