### **2. Memory Processing Pipeline**

**Step 1: Conscious Logging**
- Roger's conscious layer logs conversations to `conscious/YYYY-MM-DD-HH.md`
- Raw, messy, real-time conversation capture

**Step 2: Subconscious Processing (Heartbeat)**
- Every 5-10 minutes, subconscious wakes up
- Scans `conscious/` directory for new logs
- Extracts metadata (importance, tags, context)
- Distills content into key points
- Saves as structured `.qmd` with YAML frontmatter
- Deletes processed conscious logs
- Updates `MEMORY.md` with significant insights

**Step 3: Fast Recall**
- Metadata enables instant search: `importance: >7`, `tags: [breakthrough]`
- No need to parse full content
- Conditional includes reduce token usage
